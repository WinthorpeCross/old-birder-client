Technology

This page summarises the technology that is used to create Birder.

See the developer page for information about the creator.

The Technology Stack

Here is a summary of the technology used by Birder:

Client

  • Angular 131 and TypeScript single-page application
  • Bootstrap and Angular Material components libraries.

Server

  • ASP.NET 62 and C# REST API
  • Azure 'serverless' SQL Database

Deployment

  • Hosted on the Azure cloud platform in Docker containers
1 Since 10th November 2021. Formerly Angular 6 — 12.
2 Since 11th November 2021. Formerly .NET Core versions 2.2, 3.0, 3.1, and .NET 5.
